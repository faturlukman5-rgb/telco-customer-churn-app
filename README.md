# 📱 Aplikasi Prediksi Retensi Pelanggan (Telco Customer Churn)

Proyek ini merupakan implementasi Machine Learning untuk memprediksi probabilitas pelanggan perusahaan telekomunikasi yang berpotensi berhenti berlangganan (*churn*) secara *real-time*.
---

## 🚀 Tautan Projek
* **Link Deployment Streamlit:** [Buka Aplikasi Web]( https://telco-customer-churn-app-gjxvas4se6krfxotvodjbz.streamlit.app/)
* **Repository GitHub:** [Kunjungi Repository](https://github.com/faturlukman5-rgb/telco-customer-churn-app)

---

## 🛠️ Metodologi & Teknik
* **Bahasa Pemrograman:** Python
* **Algoritma Model:** Logistic Regression (Regresi Logistik)
* **Pustaka Utama:** Streamlit, Scikit-Learn, Pandas, Joblib

---

## 📁 Struktur File Repository
* `app.py` - Kode utama antarmuka aplikasi web berbasis Streamlit.
* `requirements.txt` - Daftar pustaka (dependencies) beserta versi yang dibutuhkan server.
* `best_model_lr.pkl` - Berkas model Logistic Regression yang telah dilatih.
* `scaler.pkl` - Berkas normalisasi data (StandardScaler).
